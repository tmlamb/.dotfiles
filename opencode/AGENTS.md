# Tom's Agent Instructions

These are common instructions for agents across all scenarios.

## General Guidelines

- Never use the em dash (—) in your responses. Use a hyphen (-) instead.
- When writing commit messages, NEVER auto-add your agent name as co-author, or
  any other metadata.
- Never modify auto-generated files unless explicitly instructed to do so.
- When making technical decisions, do not give much weight to development cost.
  Instead prioritize quality, simplicity, rubustness, scalability, and long term
  maintainability.
